Difference between Promise and Observable

Promise

            1. asyn 
            2. Returns single value 
                    resolved or rejected
            3. Promise can't be cancelled
            4. Promises are executed eagerly.  Eagar: The Promise callback will execute right away at the constructor level.

Observable

            1. asyn / sync
            2. Returns stream of values
            3. observable  can be cancelled
            4. Observables are executed lazily. Lazy: The Producer function will only trigger after there is a subscription created for that Observable. Otherwise, it will stay idle.
